# Login Template code

## React Client
### Requirements
- Yarn
- TypeScript
- Port 3000 open


1. run 

```
yarn
```

3. run
```
tsc build
```

4. run
```
yarn start
```

## Express server

### Requirements
- MySQL
- Yarn
- TypeScript
- NodeJS
- Port 5000 open

1. You need a .env file at the root of the server folder e.g.
```
SQL_DB=logintemplate
SQL_USERNAME=root
SQL_PASSWORD=root
SQL_HOST=localhost
JWT_SECRET=9eJczs2DcuasYRy2kgHx33%mVH$Frw8NCe6J*J2B
```

2. run 

```
yarn
```

3. run
```
tsc build
```

4. run
```
node src/index.js
```
